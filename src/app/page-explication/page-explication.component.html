Explications

Le but est de comparer le nombre d'operation réaliser par Angular, avec et sans le mot clé Keyword

Doc Angular : https://angular.io/api/common/NgIf#storing-a-conditional-result-in-a-variable

En regardant la console, vous verez que le nombre d'opération est bien plus conséquent sans l'utilisation du AS

Actions :
  • Ouvrez la console
  • Navigez dans le menu en haut à droite
  • Constatez

Résultat

Les nombres diffèrent :

Pour le composant PageSansKeywordComponent, le nombre d'opération est 304

Pour le composant PageAvecKeywordComponent, le nombre d'opération est 92

Alors que le rendu visuel est le meme et qu'il manipule les memes données, seul l'implementation du composant person change
